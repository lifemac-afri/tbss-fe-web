# TorchBearers Books and Stationery Services — Component Library

Welcome to the official TBSS design system documentation. This Storybook captures every reusable UI primitive and feature component used across the platform.

---

## Brand Colors

| Token | Hex | Usage |
|-------|-----|-------|
| **Primary** | `#F46B03` | CTAs, buttons, active states |
| **Primary Dark** | `#C15300` | Hover states |
| **Secondary** | `#1C25F2` | Links, highlights, secondary actions |
| **Success** | `#16A34A` | Confirmation, in-stock |
| **Warning** | `#D97706` | Alerts, promotions |
| **Error** | `#DC2626` | Validation errors, danger actions |

## Typography

- **Display / Logo**: Aclonica
- **UI / Body**: Poppins (300 · 400 · 500 · 600 · 700)

## Component Categories

- **Primitives** — Button, Input, Badge, Spinner, Skeleton
- **Overlays** — Modal, Toast
- **Product** — BookCard

---

> Components are built with Tailwind CSS v3 and require no external UI library.
